# Installatiom
- Run command :
```
mkdir /coolq4s-scripts
cd /coolq4s-scripts

git clone https://github.com/coolq4s/server-lnx.git 
mkdir /script 
mv server-lnx/Watchdog/varlog-troubleshot.sh /script/varlog-troubleshot.sh
chmod +x server-lnx/automation-scripts/var_log/Install-Varlog.sh
bash server-lnx/automation-scripts/var_log/Install-Varlog.sh
```
- Done
